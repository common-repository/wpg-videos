A simple and responsive YouTube video gallery player with thumbnails.

== Description ==
Display a responsive video gallery of YouTube videos. Options include:

- Easily change the play order of videos
- Enable autoplay for the first video
- Set the highlight color of the active video\'s thumbnail

== Installation ==
- Install YouTube Responsive Gallery either via the WordPress.org plugin directory, or by uploading the files to your server.
- Activate the plugin.

== Frequently Asked Questions ==
**What player options are available?**

You can set the color of the active video's thumbnail, and also enaable/disable autoplaying of the first video.

**What does the Premium version add?**

The Premium version allows you to link to MP4 videos in the WordPress media library or externally hosted.
